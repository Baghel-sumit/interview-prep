
# [Apnagram](https://apnagram.onrender.com)

A web application which is a clone of a social media application.


## Steps to start with project

### 1. To run the project

```bash
    npm install
    npm run dev
```

### 2. To run lint command

```bash
    npm run lint
    npm run lint:fix
```
